# CodingAgentDemo

## 🎮 4 Gewinnt (Connect Four)

Ein schönes Browser-basiertes 4-Gewinnt-Spiel mit Animationen!

### Features

- 🎨 Modernes, responsives Design mit Farbverläufen
- ✨ Animierte Spielsteine mit Drop-Effekt
- 🏆 Automatische Gewinnerkennung
- 💫 Pulsierende Animation für Gewinnkombinationen
- 🔄 Unentschieden-Erkennung
- 📱 Mobilfreundlich

### Spielen

Das Spiel ist automatisch über GitHub Pages verfügbar. Klicke einfach auf eine Spalte, um deinen Spielstein zu platzieren!

**Ziel:** Verbinde vier Spielsteine in einer Reihe (horizontal, vertikal oder diagonal).

### Lokales Testen

1. Klone das Repository
2. Öffne `index.html` in einem Browser
3. Viel Spaß beim Spielen!

### Deployment

Das Spiel wird automatisch über GitHub Actions auf GitHub Pages bereitgestellt, wenn Änderungen in den `main` Branch gepusht werden.
